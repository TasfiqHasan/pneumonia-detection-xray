# Pneumonia Detection from Chest X-Ray Images

## Overview

This project uses Convolutional Neural Networks (CNNs) to classify chest X-ray images as Normal or Pneumonia.

## Technologies Used

* Python
* TensorFlow/Keras
* OpenCV
* NumPy
* Matplotlib

## Dataset

Chest X-ray dataset containing pediatric patient images.

## Model

* CNN Architecture
* Data Augmentation
* Early Stopping

## Results

Model Accuracy: 95%+
